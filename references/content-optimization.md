# Content Rewriting & Optimization (Reworker)

Goal: Revitalize existing content to improve rankings, freshness, and engagement.

## When to Rewrite vs. When to Refresh

**Full Rewrite (Rework):** Use when:
- Article is more than 2 years old with outdated information
- Rankings have dropped significantly (lost top 10 position)
- Original content is thin (under 1,000 words) and needs expansion
- Search intent has shifted (what people want from this keyword has changed)
- Competitor articles are substantially better

**Content Refresh:** Use when:
- Statistics, dates, or product versions need updating
- Adding a new section to match current best practices
- Improving title/meta description for higher CTR
- Adding internal links to newer content

---

## Full Rewrite Process

### Step 1: Audit the Existing Article

Gather this information:
- Current ranking position (ask user, or note "unknown")
- Current word count
- Publish date
- Core topic / primary keyword
- What sections already exist

### Step 2: Analyze Top-Ranking Competitors

Use web search to find the top 3-5 results for the target keyword.
For each competitor, note:
- Word count (estimate)
- Headings used (H2 structure)
- Topics they cover that the original article misses
- Content formats used (tables, videos, tools, calculators)
- Freshness signals (dates, recent data)

### Step 3: Identify Gaps and Improvements

Create a gap analysis:
| What Original Has | What Competitors Have | Action |
|---|---|---|
| Basic definition | Deep how-to | Expand section |
| 2021 statistics | 2024 data | Update stats |
| No FAQ | FAQ with 7 questions | Add FAQ section |
| No comparison table | Feature comparison | Add table |

### Step 4: Rewrite with Enhancements

Apply all improvements from gap analysis PLUS:
- Update all dates, statistics, and product versions
- Improve the introduction (hook + promise within first 100 words)
- Add/expand sections that competitors rank for
- Add FAQ section if missing
- Add a comparison table if applicable
- Strengthen internal linking (add 2-3 new internal links)
- Rewrite the conclusion with a clear CTA

### Step 5: On-Page SEO Pass

- Confirm primary keyword in: title, H1, first 100 words, meta description
- Add secondary keywords identified from current SERP
- Check and improve heading hierarchy
- Add/update image alt text
- Update URL slug if it's non-descriptive (coordinate with user — 301 redirect needed)

---

## Quick Refresh Process (for minor updates)

1. Update the publication date and add "Updated [Month Year]" to title
2. Replace all statistics older than 18 months with current data
3. Remove any references to discontinued products/services
4. Add 1-2 new sections covering subtopics competitors now rank for
5. Add internal links to any new relevant content published since original
6. Improve meta description if CTR is low
7. Add FAQ schema if none exists

---

## CTR Optimization (Title & Meta Improvement)

Use this when rankings are fine but click-through rate is low.

### Title Optimization Formulas
- "[Number] Best [Product] for [Audience] ([Year])" — listicles
- "How to [Achieve Goal] in [Timeframe]" — tutorials
- "[Topic A] vs [Topic B]: Which Is Better in [Year]?" — comparisons
- "The [Adjective] Guide to [Topic] for [Audience]" — comprehensive guides
- "Why [Common Belief] Is Wrong (And What to Do Instead)" — contrarian

### Meta Description Formula
"[Benefit statement]. Learn [specific thing they'll get]. [Light CTA]."

Example: "Stop wasting money on the wrong podcast mic. We tested 15 options to find the best microphones for every budget. See our top picks."

## Content Score Checklist (grade before and after rework)

Score 1 point for each:
- [ ] Primary keyword in title
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in meta description
- [ ] Word count 2,000+ (or 1,500+ for commercial pages)
- [ ] At least one H2 with secondary keyword
- [ ] FAQ section present
- [ ] Comparison table or data visualization present
- [ ] 3+ internal links
- [ ] 1+ external authoritative link
- [ ] Images with keyword alt text
- [ ] Schema markup applied

Score 10/12: Good | 11/12: Great | 12/12: Optimized
